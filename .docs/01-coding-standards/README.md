# コーディング規約（Coding Standards）

このディレクトリには、コーディング規約に関する知識を格納しています。

**カテゴリID**: `01-coding-standards`
**更新頻度**: 中頻度（規約追加・変更時）
**参照優先度**: 高（コーディング作業時は必須）

## ファイル一覧

| 用途 | ファイル | 内容 |
|------|---------|------|
| 変数名やメソッド名を決める時 | [命名規則](./naming-conventions.md) | 変数名、メソッド名、型名の命名規則 |
| コード構造やルールを確認する時 | [コード構造](./code-structure.md) | ネスト、責務の分離など |
| 型アノテーションを書く時 | [型アノテーション](./type-annotations.md) | 関数・変数の型アノテーションの書き方 |
| 型エイリアスを使うべきか判断する時 | [型エイリアス](./type-aliases.md) | 型エイリアスの活用方法 |
| Docstringを書く時 | [Docstring品質ガイド](./docstring-guide.md) | Docstringの書き方 |

## 参照方法

コーディング作業時は、該当するファイルを参照してください。すべての規約を一度に参照する必要はありません。
